# PersistentHomology-NeuralComplexity-
Reproducible pipeline for persistent homology analysis of neural dynamics during CO₂ euthanasia using MATLAB, Python, and Ripser.
#
# Data

The complete experimental recordings are not publicly distributed because they are part of ongoing research.

This repository includes example datasets that reproduce the complete analysis pipeline.

To request access to the original recordings, please contact the author: khitam Aqel at: kza4256@mavs.uta.edu.
